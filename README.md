# AlphaOrbit Worldwide Ltd - Corporate Website

This repository contains the official, high-performance corporate web platform for **AlphaOrbit Worldwide Ltd**. The website serves as a unified digital gateway showcasing the company's five core commercial pillars across international trade, logistics, and consultancy services.

## 🚀 Live Demonstration
The website is automatically built and hosted using GitHub Pages. 
* **Live Link:** [https://bobmanuel-duabo.github.io/alphaorbit/](https://bobmanuel-duabo.github.io/alphaorbit/)

---

## 🏢 Core Business Pillars Built-In
The platform structures and handles corporate inquiries for five distinct service tracks:
1. **🌾 Agro-Commodities & Seafood Export** – Foodservice management, foodstuffs distribution, and global cold-chain seafood supply.
2. **🔥 Sustainable Energy** – Premium high-caloric industrial and commercial hardwood charcoal export operations.
3. **🪡 Fashion & Accessories** – Modern international apparel design, logistics, and textile distribution.
4. **⚓ Global Logistics** – End-to-end freight forwarding, customs clearance, warehousing, and trade fulfillment.
5. **🤝 Caregiver Contracting & Consultancy** – Professional healthcare placement, contract workforce administration, and HR compliance consulting.

---

## 🛠️ Tech Stack & Architecture
* **Frontend:** Clean, semantic **HTML5**, custom responsive **CSS3** variables (derived directly from the corporate brand asset colors), and modern **JavaScript** for form logic.
* **Local Development Environment:** Powered by a lightweight **Python** development server layer (`app.py`) ensuring clean path routing and testing.
* **Hosting & Deployment:** Continuous Integration/Continuous Deployment (CI/CD) automated via **GitHub Pages**.
